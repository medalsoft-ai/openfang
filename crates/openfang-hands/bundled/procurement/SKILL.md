# Procurement Expert Knowledge — 商品采购调研专业指南

## 采购调研 SOP 框架

### 标准调研流程
```
需求确认 → 市场扫描 → 品牌筛选 → 深度调研 → 数据分析 → 报告输出
```

### 各阶段要点

#### 1. 需求确认
- 明确采购品类（一级分类 → 二级分类 → 具体SKU）
- 确定调研目的（新品引进/供应商评估/价格谈判/合同续签）
- 收集公司内部数据（历史采购量、现有供应商、销售数据）
- 设定预算范围和质量标准

#### 2. 市场扫描
- 了解行业整体规模和市场趋势
- 识别主要品牌和新兴品牌
- 了解渠道结构（线上/线下/全渠道）
- 了解行业淡旺季规律

#### 3. 品牌筛选
- 按市场份额筛选 Top 10
- 按价格带筛选（高端/中端/经济型）
- 按渠道适配度筛选
- 按公司现有商品结构筛选（避免同质化）

#### 4. 深度调研
对筛选出的 3-5 个重点品牌进行深度调研。

## 饮料行业调研要点

### 品类细分
| 品类 | 代表品牌 | 价格带 | 主要渠道 |
|------|---------|--------|---------|
| 瓶装水 | 农夫山泉、怡宝、娃哈哈 | 1-3元 | 全渠道 |
| 碳酸饮料 | 可口可乐、百事可乐 | 3-6元 | 便利店、超市 |
| 茶饮料 | 康师傅、统一、农夫山泉 | 3-5元 | 超市、便利店 |
| 果汁 | 农夫果园、汇源、味全 | 5-10元 | 超市、生鲜 |
| 能量饮料 | 红牛、东鹏特饮、乐虎 | 6-10元 | 便利店、电商 |
| 乳饮料 | 伊利、蒙牛、旺仔 | 4-8元 | 超市、送水站 |

### 关键数据点
1. **出厂价 vs 终端价**：计算渠道利润空间
2. **返利政策**：季度返利、年终返利结构
3. **起订量**：最低起订量和配送周期
4. **账期**：付款周期（现结/周结/月结）
5. **退换货**：滞销品处理政策

### 质量信息源
- 国家市场监督管理总局：https://samr.saic.gov.cn
- 各省食品安全抽检信息公示
- 中国消费者协会投诉数据
- 海关进出口数据（进出口品牌）

### 价格信息源
- 1688 阿里巴巴批发网（批发参考价）
- 京东企业购（企业采购价）
- 慢慢买比价网（历史价格走势）
- 中国价格信息网

## 数据分析与决策框架

### 品牌评估矩阵
```
总分 = 口碑分(25%) + 销量分(25%) + 价格分(20%) + 质量分(15%) + 供应分(15%)
```

| 指标 | 评估方法 | 数据来源 |
|------|---------|---------|
| 口碑 | 电商评分 + 社交媒体情感分析 | 京东/淘宝评分、小红书/微博 |
| 销量 | 市场份额 + 增长率 | 行业报告、电商榜单 |
| 价格 | 毛利率空间 + 稳定性 | 1688、经销商询价 |
| 质量 | 抽检记录 + 认证情况 | 食药监局、企业官网 |
| 供应 | 配送能力 + 响应速度 | 企业官网、历史合作 |

### 采购风险评估
- **品牌风险**：品牌知名度下降、品牌危机事件
- **供应风险**：产能不足、原材料波动
- **价格风险**：价格大幅波动、供应商垄断
- **质量风险**：食品安全事件、质量投诉
- **账期风险**：资金占用、付款条件不利

## 报告撰写规范

### 报告结构
1. 执行摘要（1页）
2. 市场概况（品类规模、趋势、渠道）
3. 品牌调研详情（各品牌分析）
4. 价格分析（历史走势、比价）
5. 质量评估（抽检、认证）
6. 采购建议（推荐品牌、建议价格、风险提示）
7. 附录（数据来源、调研记录）

### 数据可视化建议
- 销量排行：横向柱状图
- 价格走势：折线图（12个月）
- 市场份额：饼图或环形图
- 品牌对比：雷达图
- 推荐决策：决策矩阵热力图

### 报告质量检查清单
- [ ] 所有数据有明确来源
- [ ] 价格数据标注采集时间
- [ ] 质检信息来自官方渠道
- [ ] 建议基于数据而非主观判断
- [ ] 风险提示明确标注
- [ ] 报告格式专业规范
