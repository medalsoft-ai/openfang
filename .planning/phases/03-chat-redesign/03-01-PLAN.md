## Key Design Tokens from Layout

### Colors
- Primary accent: `#8B5CF6` (violet-500)
- Background gradient: `from-white via-purple-50/50 to-violet-100/30`
- Surface: white with `bg-white/80 backdrop-blur-xl`
- Border: `border-purple-100`
- User message bg: `bg-gradient-to-br from-violet-100 to-purple-100`
- AI message bg: `bg-white`

### Shadows (Claymorphism)
- Standard: `shadow-[0_2px_8px_rgba(139,92,246,0.08)]`
- Hover: `shadow-[0_4px_12px_rgba(139,92,246,0.12)]`
- Active/pressed: `shadow-[inset_0_2px_4px_rgba(139,92,246,0.1)]`
- Input focus: `shadow-[0_4px_20px_rgba(139,92,246,0.2)]`

### Typography
- Body: 15px, leading-relaxed
- Small: 13px
- Caption: 12px
- Font: Inter (system-ui fallback)

### Border Radius
- Message bubbles: `rounded-2xl` with speech bubble effect (`rounded-tr-sm` for user, `rounded-tl-sm` for AI)
- Cards: `rounded-xl`
- Buttons: `rounded-xl`

### Animations
- Message entry: spring physics (stiffness: 200, damping: 20)
- Stagger children: 0.05s delay
- Hover transitions: 150-200ms
- Page transition: already in `lib/animations.ts`
